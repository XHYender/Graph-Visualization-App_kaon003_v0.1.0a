请直接运行main.py
请确保你使用python3.11打开(3.X应该都行吧)，并安装了tkinter库!

你可以通过修改程序根目录文件下的save_number.txt的内容为任意数字以改变你想要读写的存档编号。0是演示存档，1是示例存档。

如果需要查看演示存档，请在不做任何更改的情况下直接按Ctrl+R

=====文件夹说明=====
.\assets	必要性文件，存放了程序图标
.\saves	必要性文件，存放了所有存档
.\developer	非必要文件，存放了项目的历史可运行版本、开发日志、内置存档
.\main.py		主程序，直接运行以打开程序
.\README.txt	此文档。
.\save_number.txt	读写存档编号，当前版本一次运行只能读写一个存档


=======按键=======
版本0.1a
<ESC>关闭窗口；关闭编辑界面(自动保存文本信息)
<S>切换场景：['limbs','nodes']
<Space>'limbs'场景下切换test limb运动模式：['Fixed','Instant','SOS']
		'nodes'场景下切换nodes模式:['Fixed','Physics']
<F3>打开/关闭F3菜单
<M>'nodes'场景下在F3菜单显示/切换邻接图

<C>鼠标处创建节点
<D>删除选中节点/边；隐藏文本框
<W>未选中时更新带权邻接图
<F2>鼠标悬停在文本框上的时候打开文本编辑界面
<T>鼠标悬停在节点或文本框上的时候打开文本编辑界面；打开编辑界面时再按一次则关闭编辑界面
<Ctrl+T>当focus在编辑弹窗时可以用于关闭编辑界面

<右键>空白处：<单击>创建节点；
	   节点：<拖拽>至另一节点以创建线；
	   线：<单击>切换线的有向/无向

<左键>节点：<单击拖拽>节点，<长按后释放>点亮/熄灭节点，<双击>显示/隐藏文本框；
	   线：<单击拖拽>线，<双击>改变有向线的方向；
	   文本框：<单击拖拽>文本框，<双击>隐藏文本框

<中键>空白处：<单击拖拽>画布
	   节点：<单击>以聚焦至节点
	   有向线：<单击>以聚焦至有向线指向的节点

<ctrl+S>保存
<ctrl+R>读取（你可以通过修改.\main.py中主函数中的Application类的save_id参数以改变读写的存档.）
